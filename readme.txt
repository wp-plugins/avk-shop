AVK-Shop allows you in automatic mode to sell of items from your website.

== Description ==

= ENG = 

AVK-Shop - a plug of web shop in your WordPress, connects of payment systems, is ideal for people who have a need to sell any files on its web site (video tutorials, photographs, programs, etc.). The plugin can run on only that WordPress installed and the site is already having their records and pages.

== Installation ==

1. Upload `avkshopweb20` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. Set and keep all options on the page plugin AVK-Shop &gt; General settings
4. Go to page AVK-Shop &gt; Tools and create a database table (table data required for the plugin)
5. That's all!)
> <h3>Attention!!!</h3><p>Create a table in the database should only be after configure and save all of the options on the page AVK-Shop &gt; General settings</p>
